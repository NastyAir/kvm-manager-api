KVM Manager API

Project Overview

The KVM Manager API provides a robust backend service for managing KVM virtual machines. It is designed to be integrated with various front-end applications to offer a complete virtualization management solution.

Features

  • Create and destroy virtual machines
  • Replicate existing virtual machines
  • Manage virtual machine configurations
  • Monitor virtual machine status

Installation and Usage

  1. Clone the repository: git clone https://github.com/NastyAir/kvm-manager-api.git
  2. Navigate to the project directory: cd kvm-manager-api
  3. Build the project using Maven: mvn clean install
  4. Run the application: java -jar target/kvm-manager-api.jar
